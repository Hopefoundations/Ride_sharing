# Taxi Booking Simulation System

## Overview

The Taxi Booking Simulation System is a console-based Java application developed using Core Java concepts. The system allows users to view available taxis, book taxis, make payments, track taxi status, and cancel bookings. It simulates a real-world taxi booking process while demonstrating Object-Oriented Programming principles.

## Features

* View available taxis and their fare rates
* Book taxis based on availability
* Prevent duplicate bookings
* Make payments using UPI or Cash
* Track taxi arrival status
* Cancel existing bookings
* View taxi booking status

Compile the program:

```bash

```

Run the application:

